# Claudia Villegas — Photography Website

GitHub Pages-ready static website for Claudia Villegas photography and videography.

## Current homepage focus

The homepage is designed to show the photographer's work first through a visual portfolio mosaic.

Current visible categories:

- Bodas
- XV años
- 3 años
- Bautizo
- Newborn

Drone and video are still included as a service section, but the visual placeholder remains until real drone/video thumbnails are ready.

## Main files

- `index.html` — page structure and content
- `styles.css` — design and responsive layout
- `script.js` — mobile menu and scroll reveal behavior
- `assets/images/` — portfolio and portrait images

## Before publishing

Update these links in `index.html`:

- Reservation button: currently `https://www.google.com`
- Contact form endpoint: currently `https://formspree.io/f/your-form-id`

## Publishing with GitHub Pages

1. Create a new GitHub repository.
2. Upload the files from this folder.
3. Go to **Settings → Pages**.
4. Choose **Deploy from a branch**.
5. Select the `main` branch and `/root` folder.
6. Save.
